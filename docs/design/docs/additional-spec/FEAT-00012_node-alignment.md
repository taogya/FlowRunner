# FEAT-00012: ノード整列（Align X / Align Y）

- 対応する RD 参照: RD-01 §10 #9（フロー編集 UX 向上）

## 概要

選択中の複数ノードを X 座標または Y 座標で揃える整列機能を追加する。既存の Auto Layout（dagre ベース）とは独立した簡易整列。

## スコープ

- **対象:** 2 つ以上選択されたノード
- **Align X (FEAT-00012-002001):** 選択ノードの X 座標を平均値に統一（縦一列に揃える）
- **Align Y (FEAT-00012-002002):** 選択ノードの Y 座標を平均値に統一（横一列に揃える）
- **スコープ外:** 等間隔配置、グリッドスナップ

## 仕様

### §2.1 整列ロジック (FEAT-00012-002001)

- 選択ノードが 2 未満の場合は何もしない
- pushState で undo 用に現在状態を保存してからノード位置を更新
- X 整列: 全選択ノードの X 平均値を算出 → 全選択ノードの X を平均値に設定
- Y 整列: 全選択ノードの Y 平均値を算出 → 全選択ノードの Y を平均値に設定

### §2.2 UI 統合 (FEAT-00012-002002)

- Toolbar に ⫏ (Align X) と ⫐ (Align Y) ボタンを追加
- 選択ノードが 2 未満のとき disabled
- 既存の Auto Layout ボタン (⇶) の右隣に配置

## 影響範囲

- `src/webview/components/Toolbar.tsx` — props 追加 (onAlignX, onAlignY, hasSelection)
- `src/webview/components/FlowEditorApp.tsx` — handleAlignX, handleAlignY ハンドラ追加

## テスト方針

- 2 ノード以上の選択状態で X/Y 平均値計算の正しさを検証
- 1 ノード以下では何も起きないことを検証
- undo 用 pushState が呼ばれることを検証
