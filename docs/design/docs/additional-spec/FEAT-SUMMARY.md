## ステータス定義

| ステータス | 説明 |
|---|---|
| 設計中 | 仕様書作成中 |
| 実装中 | コード実装中 |
| テスト中 | テスト作成・実行中 |
| 完了 | 実装・テスト完了 |
| 廃止 | 仕様・実装を廃止し、履歴として保持 |
| 保留 | 一時的に作業停止 |
| 将来対応 | 仕様は記録済みだが、現行リリース対象外 |
